A comprehensive Human Resource Management System featuring role-based multi-panel access for administrators, HR managers, and employees. The system streamlines employee management, attendance tracking, leave requests, payroll processing, and performance reviews. ## **Key Features Delivered**

* Multi-panel authentication system (Admin, HR, Employee portals)
* Complete employee lifecycle management with department/position hierarchy
* Real-time attendance tracking with check-in/out functionality
* Leave management system with approval workflows
* Automated payroll generation via Laravel commands and scheduled jobs
* Performance review system with calculated ratings
* Dynamic settings page for currency, date formats, and system configurations
* Queue-based background processing for scalability
* Role-based access control with Filament Shield integration
